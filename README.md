## üéØ Overview

WatiqAI is a cutting-edge FinTech solution designed to combat document fraud and streamline Know Your Customer (KYC) processes for financial institutions in the Moroccan market. Leveraging advanced Artificial Intelligence, WatiqAI provides real-time, highly accurate verification of identity documents.

## ‚ú® Features

- **70% Cost Reduction** - Significantly lower manual verification overhead
- **99%+ Accuracy** - Multi-modal AI engine for unmatched fraud detection
- **<5s Verification** - From 15-30 minutes to near-instant processing
- **100% Compliance** - Aligned with Bank Al-Maghrib regulations

## üöÄ Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Local development server (optional)

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/watiqai.git
   cd watiqai
   ```

2. **Open the website**
   - Simply open `index.html` in your browser, or
   - Use a local server:
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js (http-server)
     npx http-server
     
     # Using VS Code Live Server extension
     # Right-click index.html ‚Üí Open with Live Server
     ```

3. **View the site**
   - Navigate to `http://localhost:8000` (or your server's port)
